## 🏁 Como executar o projeto

```bash
# Clone este repositório
$ git clone https://github.com/thegabrielrodrigues/imhere.git

# Acesse a pasta do projeto
$ cd imhere

# Instale o yarn caso não esteja instalado
$ npm install -g yarn

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start
```
